Windows 1.0, 2.0 e 3.0

A evolução gráfica do Windows é uma jornada que transforma o computador de uma ferramenta técnica em um ambiente pessoal e intuitivo. Nos seus primeiros passos, com o Windows 1, 2 e 3, o sistema funcionava como uma casca visual para o MS-DOS. Nessa fase, a interface era rudimentar e limitada, focada em apresentar janelas simples que mal podiam se sobrepor no início. O grande avanço foi o Windows 3.1, que trouxe ícones mais nítidos e fontes escaláveis, permitindo que o usuário começasse a enxergar o monitor não como uma tela de códigos, mas como uma mesa de trabalho virtual com pastas e arquivos.
